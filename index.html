自我介紹

我是一個對生活充滿好奇心的普通人，平時喜歡透過學習新事物來拓展自己的視野。工作之餘，習慣用閱讀和音樂放鬆心情。不擅長華麗的自我包裝，但做事認真、重視誠信。希望透過這個網站，慢慢記錄下自己的想法與成長。我有一個抹布狗鄰居、一個優比狗鄰居、一個長毛狗鄰居。

簡歷

  • 工作內容涵蓋機電整合系統設計、自動化量測系統開發、控制流程最佳化與數據分析。
  • 熟悉 MATLAB、Python 與 C/C++ 程式開發，具備系統整合、測試驗證與跨領域工程合作經驗。

專案與工程經驗

工研院

自動化介電量測系統開發

整合共振腔量測治具、網路分析儀（VNA）與馬達載台，使用 MATLAB 開發控制程式，建立自動化介電常數量測系統。

Python 實驗資料管理工具

開發 Python 應用程式，自動讀取量測檔案、分類資料並生成 Excel 報表，提高資料管理效率與可靠度。

LTCC 材料與元件製程

參與低溫共燒陶瓷材料開發與元件製程實驗，包含粉體分析、熱分析、電性量測與微結構觀測。

自主學習專案

工廠薪資計算系統

以 Python + Flask + SQLite 開發單機薪資管理系統，支援出勤匯入、加班費自動計算與勞健保試算，並可匯出薪資條 PDF 及銀行轉帳 Excel。

RC Car 遙控車製作

自製遙控車，涵蓋底盤結構設計、馬達驅動電路配置與遙控系統整合，並撰寫完整技術報告記錄設計流程與測試結果。

ESP32 室內外天氣站

以 ESP32 搭配 DHT11 感測器與 Open-Meteo API，即時顯示室內外溫濕度與降雨預報，內建 Web 設定介面，支援 Flash 儲存與 AP 備援模式。

個人網站

以原生 HTML / CSS / JavaScript 打造的個人作品集網站，透過 GitHub 版本控制並部署於 Vercel，收錄作品、技術筆記與生活雜記。
